# Selected Journal Decision

Primary target journal: PLOS ONE.

Backup journals: JAMIA Open; BMC Medical Informatics and Decision Making.

Rationale: PLOS ONE is broad enough for a reproducible public-data benchmark, explicitly supports methods/software/database-type work when utility, validation, and availability are demonstrated, and provides LaTeX guidance. The project does not claim clinical deployment readiness, which fits PLOS ONE's emphasis on methodological rigor over exaggerated novelty.

Main risk: the paper must make clear that this is a benchmark and evidence-triage component, not a clinical decision-support tool. The manuscript therefore foregrounds public data, calibration, external validation, reproducibility, and limitations.
