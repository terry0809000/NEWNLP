# Literature Search Protocol

Date searched: 2026-05-24.

Sources: PubMed, PubMed Central, ACL Anthology, arXiv, PMLR, journal/publisher pages, official dataset repositories, and official journal pages. Searches targeted biomedical abstract sentence classification, PICO extraction, PubMed RCT datasets, calibration, external validation, reproducibility, and responsible healthcare AI reporting.

Inclusion criteria: public dataset or methodological papers directly relevant to biomedical sentence classification, CPU-feasible classical/neural baselines, calibration assessment, external validation, or reporting. Exclusion criteria: papers without accessible citation metadata, unrelated transformer-only systems, restricted clinical datasets, and non-primary blog or forum sources.

No performance numbers from the literature were copied into this project unless they were needed only as qualitative background. Experimental claims in the manuscript use metrics generated locally.
