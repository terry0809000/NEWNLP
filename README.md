# NEWNLP

Submission-ready healthcare NLP manuscript package and supporting project documents.

## Contents

- `submission_ready_latex/`: complete LaTeX submission package, including `main.tex`, `references.bib`, sections, tables, figures, supplementary files, cover letter source, checklist, compile log, and compiled `main.pdf`.
- `project_outputs/10_manuscript/`: manuscript markdown files, evidence lock, submission checklist, readiness report, and cover-letter draft.
- `project_outputs/11_latex_zip/`: final LaTeX zip packages and quality/compile reports.
- `project_outputs/08_literature/`: expanded bibliography and literature-review files.
- `project_outputs/09_journal_selection/`: journal-selection documentation.
- `archives/NEWNLP_all_documents_upload.zip`: single archive containing the uploaded document set.
- `scripts/60_build_submission_package.py`: script used to regenerate the submission-ready LaTeX package from verified project outputs.

## Key Files

- Final submission zip: `project_outputs/11_latex_zip/healthcare_nlp_submission_ready_latex.zip`
- Compiled manuscript PDF: `submission_ready_latex/main.pdf`
- Manuscript readiness report: `project_outputs/10_manuscript/MANUSCRIPT_READINESS_REPORT.md`
- Evidence lock file: `project_outputs/10_manuscript/evidence_lock_file.md`

The manuscript package is aligned to PLOS ONE and contains 62 cited bibliography entries. Final author details, repository/DOI, affiliation, and submission-system declarations still require author confirmation before journal submission.
